# Tomcat Sync

Tomcat Sync 是一个用于快速同步本地项目文件到远程 Tomcat 服务器的 Go 语言工具。它通过 SSH 连接，支持增量更新、并行上传和多服务器配置管理。

## 🚀 主要特性

- **交互式 CLI**: 友好且美观的终端交互界面。
- **差异对比**: 智能对比本地与远程文件差异（基于大小和修改时间），仅同步有变动的文件。
- **并行上传**: 支持多线程并行上传，极大提升同步效率。
- **灵活配置**: 支持多服务器、多路径组合（Groups）配置。
- **远程清理**: 可选清理远程服务器上多余的文件，保持目录一致。
- **多种认证**: 支持 SSH 密码认证和密钥文件认证。

## 🛠️ 安装

### 从源码编译

确保你已安装 Go 1.20 或更高版本。

```bash
git clone <repository-url>
cd tomcat-sync
go build -o tomcat-sync.exe
```

## 📖 使用指南

1. **启动程序**:
   直接运行编译好的二进制文件：
   ```bash
   ./tomcat-sync.exe
   ```

2. **配置服务器**:
   - 第一次运行时，程序会引导你添加服务器信息。
   - 配置文件默认保存在 `~/.tomcat-sync/config.yaml`。

3. **同步文件**:
   - 选择服务器。
   - 选择定义的路径组合（Group）。
   - 程序将自动执行差异对比并展示结果。
   - 确认后执行同步。

## 📁 项目结构

- `main.go`: 程序入口。
- `config/`: 配置管理逻辑。
- `ssh/`: SSH 客户端封装，处理连接与文件传输。
- `ui/`: 基于控制台的交互式界面。
- `compare/`: 核心差异对比逻辑。
- `upload/`: 文件上传执行器。
